שאלות נפוצות

האם צריך ידע בתכנות?

לא. הקורס מיועד לאנשים בלי רקע טכני. כל מה שצריך זה לדעת לכתוב בעברית — Claude Code עושה את השאר.

כמה עולה?

הקורס עצמו חינמי לגמרי. כדי להשתמש ב-Claude Code צריך מנוי Claude Pro שעולה $20 לחודש. זה ההשקעה היחידה — ובתמורה מקבלים כלי שבונה אתרים, מצגות, משחקים ואוטומציות.

האם $20 לחודש שווה את זה?

משתתפי ההדרכה דיווחו שהם חסכו שעות עבודה כל שבוע. מורה אחת אמרה: "חסכת לי המון זמן ותלמידים רבים יוכלו להנות." אם הזמן שלכם שווה משהו — זו השקעה שמחזירה את עצמה תוך ימים.

כמה זמן לוקח לסיים?

הקורס עצמאי — לומדים בקצב שלכם. רוב האנשים מסיימים תוך 2-3 שבועות בקצב של שיעור ביום. אבל אפשר גם מהר יותר.

מה אפשר לבנות עם זה?

אתרים, דפי נחיתה, מצגות, משחקים לימודיים, שאלונים, דוחות, ניוזלטרים, אוטומציות לGmail וליומן — וזה רק ההתחלה. הקורס כולל 8 כלים מוכנים לשימוש מיידי.
