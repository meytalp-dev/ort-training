איך מוסיפים סטיקרים לוואטסאפ?

  1. הורידו את הסטיקר כקובץ WebP (הכפתור הירוק)
  2. התקינו אפליקציית Sticker Maker מחנות האפליקציות
  3. פתחו את האפליקציה וצרו חבילת סטיקרים חדשה
  4. הוסיפו את קבצי ה-WebP שיצרתם לחבילה
  5. נדרשים לפחות 3 סטיקרים בחבילה
  6. לחצו "הוסף לוואטסאפ" באפליקציה
  7. הסטיקרים יופיעו במגירת הסטיקרים שלכם
טיפ: ניתן גם לשלוח את קובץ ה-WebP ישירות בוואטסאפ כקובץ מצורף, ומקבל ההודעה יוכל לשמור אותו כסטיקר ע"י לחיצה ארוכה.
