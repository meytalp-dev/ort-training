📸 קטגוריית Games – כולל Escape Games

בניית חדר בריחה — שלב אחרי שלב

1

בחרו תבנית חדר בריחה

חפשו "Escape Room" בגלריית התבניות ובחרו עיצוב שמתאים

2

כתבו את הסיפור

צרו עלילה מסקרנת לשקף הפתיחה — למה צריך "לברוח"? מה הרקע?

3

הכינו 3–4 משימות

כל משימה = שקף נפרד עם שאלה, חידה או פאזל שקשור לחומר הנלמד

4

הגדירו אינטראקטיביות

לתשובה נכונה — "Go to page" לשלב הבא, לשגויה — Tooltip עם הודעת שגיאה

5

הוסיפו רמזים

כפתורי רמז עם Tooltip או Window — התלמידים ילחצו רק אם נתקעים

6

בנו שקף קוד סופי

כל משימה נותנת חלק מהקוד. בסוף — התלמידים מרכיבים את הקוד ו"בורחים"

7

בדקו ושתפו

נסו בעצמכם, וודאו שכל הקישורים עובדים, ושלחו לתלמידים את הלינק

💡 טיפ: התחילו עם 3 משימות בלבד ובנו בהדרגה
