מדריך התקנה

CRM + אוטומציות שיווק

3 שלבים להקמת מערכת שיווק אוטומטית מלאה

1

הקמת CRM ב-Google Sheets

  1. פתחי Google Sheet חדש בשם "CRM שיווק — מיטל פלג"
  2. לחצי על Extensions → Apps Script
  3. מחקי את הקוד הקיים והדביקי את התוכן של crm-setup-script.js
  4. לחצי על Run → setupCRM
  5. אשרי הרשאות (פעם ראשונה בלבד)
  6. המתיני עד שתראי בלוג: "CRM הוקם בהצלחה!"

הגיליונות שנוצרים:

דשבורד

12 מדדים אוטומטיים — לידים, המרה, תכנים

לידים

שם, מייל, מקור, סטטוס, ציון, ניוזלטר

הדרכות

קורסים, סדנאות, תאריכים, משתתפים, משוב

תוכן

לוח פרסום, סדרות, סטטוס, ביצועים

ניוזלטר

תכני ניוזלטר שבועי — נושא, תוכן, סטטוס

חשוב! בסוף ההקמה תראי בלוג את ה-SPREADSHEET_ID — שמרי אותו לשלב הבא.
2

התקנת אוטומציות מייל

  1. באותו Google Sheet, חזרי ל-Extensions → Apps Script
  2. צרי קובץ חדש: לחצי על + → Script → שם: marketing
  3. הדביקי את התוכן של marketing-apps-script.js
  4. עדכני את CONFIG.SPREADSHEET_ID עם ה-ID מהשלב הקודם
  5. הריצי setupAllTriggers() פעם אחת
  6. אשרי הרשאות נוספות (Gmail)
var CONFIG = {
  SPREADSHEET_ID: 'הדביקי_כאן_את_ה_ID',
  ADMIN_EMAIL: 'meytalp@bethaarava.ort.org.il',
  ...
};

3

פרסום ה-Web App וחיבור לטפסים

  1. ב-Apps Script, לחצי על Deploy → New deployment
  2. בחרי Web app
  3. הגדרות:
    Execute as: Me
    Who has access: Anyone
  4. לחצי Deploy
  5. העתיקי את ה-Web App URL
  6. הדביקי את ה-URL בטופס הרישום בדף הנחיתה

דוגמה לשליחת ליד מהאתר:

fetch('APPS_SCRIPT_URL', {
  method: 'POST',
  body: JSON.stringify({
    action: 'newLead',
    name: 'שם הנרשם',
    email: 'email@example.com',
    phone: '050-1234567',
    source: 'website',
    interest: 'קורס Claude Code'
  })
});
שימי לב אחרי כל שינוי בקוד ה-Apps Script, צריך לעשות Deploy חדש:
Deploy → Manage deployments → עריכה → גרסה חדשה → Deploy
✓

צ'קליסט סיום

  • Google Sheet נוצר עם כל הגיליונות
  • SPREADSHEET_ID הועתק ל-Apps Script
  • setupAllTriggers() הורץ בהצלחה
  • Web App פורסם ויש URL
  • טופס הרישום מחובר ל-URL
  • מייל בדיקה נשלח ונקלט בגיליון
  • תזכורת פרסום התקבלה (יום למחרת 08:00)
