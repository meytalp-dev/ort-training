פתיחה והכרות
3 דקות

פתיחה קצרה ואנרגטית. מי את, מה נעשה היום, ומה הם יקבלו בסוף.

  • הצגה עצמית קצרה (30 שנ׳)
  • "היום תראו איך אני בונה משחק שלם מאפס – בלי לכתוב שורת קוד אחת"
  • "בסוף המפגש – תבינו בדיוק מה Claude Code יכול לעשות בשבילכם"

הצעה: פתחי עם שאלה – "מי פה כבר השתמש ב-ChatGPT? מי ב-Gemini?" → "מעולה, אז היום אני אראה לכם את השלב הבא."

2
למה Claude Code – ההבדל בדקה
2 דקות

הסבר חד וקצר. לא להיכנס להשוואות ארוכות – הדמו עצמו מוכיח.

  • ChatGPT / Gemini = שיחה. אתם שואלים, הם עונים
  • Claude Code = סוכן. אתם אומרים מה אתם רוצים, הוא בונה
  • "ChatGPT מסביר לכם איך לבשל. Claude Code נכנס למטבח ומבשל בשבילכם."

מקסימום 2 דקות כאן. ההסבר הזה צריך להיות חד כתער – משפט אחד לכל כלי. הדמו החי ידבר בעד עצמו.

חלק ב׳ – הגדרות ועבודה

דמו התקנה

סרטון מוכן מראש

"לפני שנתחיל לעבוד – בואו נראה איך מתקינים את Claude Code. הכנתי לכם סרטון קצר."

  1. מנוי Claude Pro – נרשמים ב-claude.ai ($20/חודש)
  2. התקנת VS Code – עורך הקוד (חינם) מ-code.visualstudio.com
  3. התקנת Claude Code – Extension מתוך VS Code
  4. חיבור חשבון – לוחצים Login ומאשרים בדפדפן
  5. בדיקה – כותבים "שלום" ורואים שקלוד עונה

שיטת העבודה: מקרינים סרטון התקנה מוכן (בלי קול) ומסבירים מעליו בלייב. מי שלא הצליח להתקין מראש – יעקוב אחרי הסרטון אחרי המפגש.

3
פתיחת Claude Code + הסבר מסך
4 דקות

פותחים את VS Code, מפעילים Claude Code, ומסבירים רק את מה שרלוונטי:

  • שורת הפקודה – פה כותבים מה רוצים (בעברית!)
  • הרשאות – כשקלוד רוצה לעשות משהו (ליצור קובץ, לקרוא, להריץ) קופצת בקשת אישור. "לוחצים Allow. זה כמו עובד שמבקש אישור לפני כל פעולה"
  • Explorer (סרגל שמאל) – פה רואים את כל הקבצים שקלוד יוצר. "כל קובץ חדש מופיע פה אוטומטית"
  • index.html – "כשקלוד בונה דף/משחק, הוא יוצר קובץ HTML. זה הקובץ שפותחים בדפדפן"
  • פתיחה בדפדפן: קליק ימני על הקובץ ב-Explorer → Reveal in File Explorer → דאבל קליק על הקובץ → נפתח בדפדפן
  • Escape = יציאה / ביטול | Ctrl+C = עצירה

לסרטון הגיבוי: הקליטי את החלק הזה מראש. הסבר מסך הוא דבר "יבש" שעובד מצוין כסרטון מוכן.
