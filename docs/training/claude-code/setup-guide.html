עיצוב ממשקים

1
היכנסו לאתר Claude

פתחו את הדפדפן וגלשו ל:
claude.ai
התחברו לחשבון שלכם
אם אין לכם חשבון -- הירשמו בחינם עם Google

2
פתחו את ההגדרות

לחצו על שם המשתמש שלכם (למעלה)
יפתח תפריט -- בחרו Settings (הגדרות)

3
היכנסו ל-Integrations

בתפריט השמאלי בחרו Integrations
כאן תראו את כל הכלים שאפשר לחבר
כך זה נראה במסך -- Settings של Claude
לא מוצאים את Integrations? לחצו על תמונת הפרופיל שלכם (בפינה הימנית העליונה) ← Settings ← חפשו בתפריט השמאלי "Integrations" (עם אייקון של שרשרת)

4
הפעילו את הכלים

לכל כלי לחצו Connect ואשרו גישה לחשבון שלכם. זה בטוח -- Claude מקבל רק את ההרשאות שאתם מאשרים.
Gmail
1. מצאו את Gmail ברשימה ולחצו על Connect
2. ייפתח חלון של Google -- בחרו את חשבון ה-Google שלכם
3. יופיע מסך הרשאות -- גללו למטה ולחצו Allow (אפשר)
4. החלון ייסגר והסטטוס ישתנה ל-Connected
נתקעתם? אם Google מבקש "Choose an account" -- בחרו את החשבון הפרטי שלכם (לא חשבון בית ספרי). אם מופיע "This app isn't verified" -- לחצו Advanced ואז Go to... (unsafe). זה בטוח.
Google Calendar
1. לחצו Connect ליד Google Calendar
2. בחרו את אותו חשבון Google שבחרתם ב-Gmail
3. לחצו Allow
טיפ: אם כבר אישרתם Gmail -- Calendar יתחבר מהר יותר כי Google כבר מכיר אתכם.
Google Drive
1. לחצו Connect ליד Google Drive
2. בחרו את אותו חשבון Google
3. לחצו Allow
מה ההרשאה נותנת? Claude יוכל לקרוא ולחפש קבצים ב-Drive שלכם. הוא לא מוחק או משנה קבצים בלי שתבקשו.
Canva
1. לחצו Connect ליד Canva
2. ייפתח דף של Canva -- התחברו לחשבון Canva שלכם (אם אין -- הירשמו בחינם)
3. לחצו Allow access
נתקעתם? Canva לפעמים מבקש להתחבר עם Google, Facebook, או מייל. בחרו את הדרך שבה נרשמתם ל-Canva. אם אתם לא זוכרים -- נסו Continue with Google.
GitHub
1. לחצו Connect ליד GitHub
2. התחברו לחשבון GitHub (אם אין -- אפשר לדלג, נפתח בהדרכה)
3. לחצו Authorize
אין לכם GitHub? זה בסדר גמור! GitHub הוא אופציונלי. אם אין לכם חשבון, פשוט דלגו -- נעזור לכם לפתוח חשבון בזמן ההדרכה.
Figma
1. לחצו Connect ליד Figma
2. התחברו לחשבון Figma (אם אין -- אפשר לדלג)
3. לחצו Allow access
אין לכם Figma? גם זה אופציונלי. Figma הוא כלי לעיצוב ממשקים. אם אתם לא מעצבים -- אפשר לדלג בלי בעיה.

!
נתקעתם? פתרונות מהירים

הכפתור Connect לא מופיע
• ודאו שאתם בעמוד Integrations (ולא ב-Settings רגיל)
• נסו לרענן את הדף (F5 או Ctrl+R)
• ודאו שאתם משתמשים בחשבון Pro (החינמי לא תומך באינטגרציות)
חלון ההרשאות נסגר בלי הצלחה
• כבו חוסם פופ-אפים (pop-up blocker) בדפדפן
• נסו דפדפן אחר (Chrome עובד הכי טוב)
• ודאו שאתם לא במצב גלישה פרטית (Incognito)
Google אומר "This app isn't verified"
• זה נורמלי! לחצו על Advanced (מתקדם) בתחתית המסך
• ואז לחצו על "Go to Claude (unsafe)"
• זה בטוח לחלוטין -- ההודעה מופיעה כי Claude עדיין בתהליך אימות מול Google
הסטטוס לא משתנה ל-Connected
• חכו כ-10 שניות ואז רענו את הדף
• אם עדיין לא מחובר -- לחצו Disconnect ואז Connect מחדש
• עדיין לא עובד? שלחו הודעה למיטל עם צילום מסך
