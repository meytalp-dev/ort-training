שאלות נפוצות

צריך לדעת קוד? ▼
בכלל לא. הכלי עובד בשיחה בעברית. את/ה אומר/ת מה צריך, והוא בונה. זה כמו לדבר עם עוזר שיודע לבנות הכל.
חובה להשתתף בכל 3 המפגשים? ▼
לא! אפשר להירשם לכל מפגש בנפרד. אבל מומלץ מאוד לעשות את כל הסדרה – כל מפגש בונה על הקודם ויחד מקבלים את התמונה המלאה.
מה ההבדל בין Claude Code לבין ChatGPT / Gemini? ▼
ChatGPT ו-Gemini כותבים טקסט. Claude Code בונה מוצרים מוגמרים – מצגות עם ניווט, שאלונים עם ציונים, משחקים אינטראקטיביים. הוא גם מתחבר לכלים שלכם ועובד בשבילכם.
זה עולה כסף? ▼
כדי לעבוד עם Claude Code צריך מנוי Pro שעולה $20 לחודש. נסביר בהדרכה הראשונה בדיוק איך נרשמים. ההדרכות עצמן ללא עלות.
צריך להביא מחשב? ▼
מומלץ! במפגש הראשון נתקין את הכלי ביחד. אם אין לכם מחשב – אפשר לצפות בהדגמות ולהתקין אחר כך.
זה רק לאנשי חינוך? ▼
לא! ההדרכות מתאימות לכולם – מורים, מנהלים, עצמאים, יזמים. נראה דוגמאות מעולם החינוך וגם מעולמות אחרים.
