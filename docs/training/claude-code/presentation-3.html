מה זה Hook?
פעולה שקורית אוטומטית כשמשהו קורה

בלי Hook

אחרי כל שינוי -- "תעשה commit"
שוכחים לשמור שינויים
חוזרים על אותן בקשות

עם Hook

Claude שומר + מעלה אוטומטית
הכל מתעדכן לבד
אפס מאמץ חוזר
Hook = "כל פעם ש-X קורה, עשה Y אוטומטית"
Hooks -- "אחרי ש..."
דברים שקורים אוטומטית אחרי פעולה
אחרי commit + push
"תפתח את הקישור החי באתר בדפדפן -- לבדוק שהכל עובד"
אחרי יצירת HTML חדש
"תוסיף אוטומטית את לוגו Learni למעלה מימין"
אחרי כל עריכת קובץ
"תבדוק שיש RTL תקין -- dir="rtl" ו-lang="he""
Hooks -- "לפני ש..."
בדיקות אוטומטיות שמונעות טעויות
לפני כל commit
"תבדוק שאין קישורים שבורים ב-HTML"
לפני כל push
"תוודא שאין סיסמאות או מפתחות API בקבצים"
כשהשיחה מתחילה
"הצג לי את המשימות הפתוחות ואת השינויים האחרונים"
Hook = שומר שלכם. עובד ברקע, מונע טעויות, חוסך זמן
