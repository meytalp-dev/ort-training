השתמשו ב-/compact כשהשיחה מתארכת כדי לשמור על זיכרון חופשי
כשהקשר של השיחה מתמלא (תוך כדי עבודה על פרויקט ארוך), הקלידו /compact וקלוד יסכם את ההחלטות החשובות וימחק את ההודעות הישנות. כך תוכלו להמשיך לעבוד בשיחה זהה ללא ירידה בביצועים.
לחצו Escape כדי לעצור מיד את תהליך הכתיבה של קלוד
אם קלוד מייצר קוד ארוך שלא נראה לכם נכון, לחצו Escape כדי להשהות את התשובה באמצע. זה מאפשר לכם לתקן את הכיוון מיד בלי להמתין לסיום, וחוסך זמן בעבודה על פרויקטים גדולים.
