תיאור הסקיל

כתבו תיאור שיעזור לקלוד להבין מתי להפעיל את הסקיל. זה השורה הראשונה בקובץ.

התיאור הזה מופיע בראש קובץ ה-SKILL.md ועוזר לקלוד להחליט מתי להפעיל את הסקיל

טיפים לתיאור טוב
  • ציינו את סוג הפלט (HTML, PDF, טקסט...)
  • ציינו את קהל היעד
  • ציינו מתי כן ומתי לא להפעיל
  • השתמשו בשפה שהמשתמש עשוי להשתמש בה
