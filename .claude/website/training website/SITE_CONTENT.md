Claude must use this content to populate the site during the initial build so that the website already contains realistic content.

All content must be written in Hebrew and RTL.

Prompts Dataset

Create at least the following prompts.

Prompt 1

Title: יצירת מערך שיעור

Prompt:

כתוב מערך שיעור בנושא [נושא] לכיתה [כיתה].
המערך צריך לכלול:

פתיח לשיעור
הסבר קצר
פעילות תלמידים
שאלות לדיון
משימת סיכום

השיעור צריך להיות בשפה פשוטה וברורה.

Prompt 2

Title: יצירת שאלות לתרגול

Prompt:

צור 10 שאלות בחירה מרובה בנושא [נושא] ברמה בינונית לכיתה [כיתה].
סמן את התשובה הנכונה.

Prompt 3

Title: דף עבודה דיפרנציאלי

Prompt:

צור דף עבודה בנושא [נושא] הכולל:

5 שאלות ברמה בסיסית
5 שאלות ברמה בינונית
3 שאלות אתגר

Prompt 4

Title: פעילות פתיחה לשיעור

Prompt:

הצע רעיון לפעילות פתיחה לשיעור בנושא [נושא] שתעורר סקרנות אצל תלמידי תיכון.

Prompt 5

Title: משוב לתלמיד

Prompt:

כתוב משוב אישי לתלמיד על העבודה הבאה:
[תיאור העבודה]

המשוב צריך להיות מעודד ומכוון לשיפור.
