# שלב 1 – ניתוח המצגת

קרא את קובץ ה-HTML של המצגת ובדוק:

### בדיקת תוכן
- [ ] האם כל שקף מכיל רעיון מרכזי אחד בלבד?
- [ ] האם המשפטים קצרים (עד 10 מילים)?
- [ ] האם השפה פשוטה ומתאימה לתלמידי תיכון?
- [ ] האם יש דוגמאות מהחיים?
- [ ] האם יש שאלות הבנה אחרי כל הסבר?
- [ ] האם יש שקף סיכום עם 3 רעיונות?

### בדיקת עיצוב
- [ ] האם נעשה שימוש ב-CSS Variables הנכונים?
- [ ] האם יש אנימציות (fadeUp, scaleIn) עם animation-delay?
- [ ] האם הפריסה משתמשת ב-grid/cards/steps?
- [ ] האם יש progress bar?
- [ ] האם הניווט עובד (חצים + touch)?
- [ ] האם התמונות משתמשות ב-`.slide-image` עם caption?

### בדיקת מבנה
- [ ] שקף כותרת עם "במצגת נלמד"
- [ ] שקף פתיחה – מה זה ולמה זה חשוב
- [ ] שקפי הקנייה מחולקים למנות קטנות
- [ ] שאלות הבנה / חידון
- [ ] סיכום

# שלב 2 – דיווח למשתמש

הצג למשתמש רשימה קצרה של:
- מה טוב במצגת
- מה צריך לשפר

ציין בדיוק אילו שקפים צריכים תיקון ולמה.

---

# שלב 3 – שיפור

בצע את השיפורים הבאים לפי הצורך:

## שיפורי תוכן

### שקפים עמוסים
- פצל שקף עמוס ל-2 שקפים
- הסר טקסט מיותר
- קצר משפטים

### שפה
- החלף שפה אקדמית בשפה פשוטה
- קצר משפטים ארוכים
- הוסף דוגמאות

### מבנה פדגוגי
- הוסף שאלות הבנה אם חסרות
- הוסף שקף סיכום אם חסר
- הוסף חידון אינטראקטיבי אם מתאים

## שיפורי עיצוב

### החלף עיצוב ישן בעיצוב חדש
- השתמש ב-CSS Variables של אורט
- הוסף אנימציות (`.anim .d1`, `.anim .d2` וכו')
- השתמש ב-`.card`, `.step-item`, `.feat-card`, `.grid-2/3/4`
- הוסף `.sec-div` לחלוקה בין סקציות

### תמונות
- עטוף תמונות ב-`.slide-image` עם `.image-caption`
- שתי תמונות זו לצד זו: `.image-pair`
- שמות קבצי תמונות **באנגלית בלבד**

### טיפים והדגשות
- השתמש ב-`.tip .tip-primary` / `.tip-green` / `.tip-peach`
- הדגש מושגים חשובים עם `<strong>` או `color:var(--primary)`

# רכיבי CSS זמינים

| Class | תיאור |
|-------|--------|
| `.card` | כרטיס לבן עם צל |
| `.icon-box` + `.ib-primary/secondary/blue/purple/green/accent/peach` | ריבוע אייקון |
| `.icon-box.sm` / `.icon-box.lg` | גדלים שונים |
| `.step-item` + `.step-num` + `.step-content` | שלב ממוספר |
| `.bullet-list` | רשימת נקודות ירוקות |
| `.tip` + `.tip-primary` / `.tip-green` / `.tip-peach` | טיפ צבעוני |
| `.feat-card` | כרטיס פיצ'ר עם אייקון |
| `.grid-2` / `.grid-3` / `.grid-4` / `.grid-5` | פריסת גריד |
| `.sec-div` | קו הפרדה דקורטיבי |
| `.slide-image` | תמונה עם סיבוב, מסגרת, hover |
| `.image-caption` | כיתוב תמונה |
| `.image-pair` | גריד שתי תמונות |
| `.escape-opt` | כפתור תשובה בחדר בריחה |

## אנימציות

| Class | אפקט |
|-------|-------|
| `.anim` | fadeUp |

| `.anim-fi` | fadeIn |
