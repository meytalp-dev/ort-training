# מדריך שפת מותג – Lerani Brand Guide

מסמך זה מגדיר את הזהות הוויזואלית של Lerani לכל התוצרים: רילס, מצגות, אינפוגרפיקות ורשתות חברתיות.

---

## הסיפור

**Lerani** = Learn + AI
"חינוך · בינה מלאכותית · הגשמת חלומות"

הזהות של Lerani היא: **מקצועית אבל נגישה, טכנולוגית אבל חמה, צעירה אבל לא ילדותית.**

---

## לוגו

### הסמל
שני עיגולים חופפים – כחול וירוק – מסמלים את **המפגש בין חינוך לטכנולוגיה**.

- עיגול שמאלי (כחול): חינוך, ידע, יציבות
- עיגול ימני (ירוק): טכנולוגיה, חדשנות, צמיחה
- אזור החפיפה: הנקודה שבה AI ופדגוגיה נפגשים

### קובץ
- SVG: `docs/assets/lerani-logo.svg`
- PNG (WhatsApp): ניתן להפיק מ-`docs/assets/logo-preview.html`

### שימוש בלוגו
- **אינטרו רילס:** אנימציית הופעה של שני העיגולים → מתחברים → "Lerani" מופיע
- **אאוטרו:** לוגו סטטי + תגית קו
- **כלל ברזל:** בכל תוצר HTML — לוגו מלא (עם כיתוב), למעלה מימין, גודל קטן, צבעים מלאים
- **מצגות:** למעלה מימין בכל שקף, width 100px
- **דפי נחיתה / אתר:** position fixed, למעלה מימין, width 120px
- **אינפוגרפיקות:** למעלה מימין + בפוטר

## מבנה רילס – תבנית

### אינטרו (0-2 שניות)
```
┌─────────────────────┐
│                     │
│     ○ ○             │  ← שני עיגולים מתקרבים
│      ◉              │  ← נחפפים
│    Lerani           │  ← טקסט מופיע
│                     │
└─────────────────────┘
```
- אנימציה: שני עיגולים נכנסים מצדדים ונחפפים
- צליל: swoosh עדין
- רקע: לבן או gradient בהיר

### Hook – שניות 2-5
```
┌─────────────────────┐
│                     │
│   🤯                 │
│                     │
│  הידעתם שאפשר       │
│  ליצור מצגת         │
│  תוך 30 שניות?      │
│                     │
│  ▶ Canva AI         │
│                     │
└─────────────────────┘
```
- טקסט גדול, ממורכז
- אימוג'י גדול למעלה
- שם הכלי בתגית צבעונית
- אנימציה: fadeUp

### תוכן – שניות 5-45
```
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │  צילום מסך של   │ │  ← 60% מהמסך
│ │  הכלי בפעולה    │ │
│ │                 │ │
│ └─────────────────┘ │
│                     │
│  ━━━━━━━━━━━━━━━━  │  ← קו הפרדה gradient
│                     │
│  שלב 1              │  ← מספר צעד + כותרת
│  לחצו על "צור חדש"  │  ← הסבר קצר
```

## סגנון תוכן

### Tone of Voice

| ✅ כן | ❌ לא |
|-------|------|
| צעיר, אנרגטי | ילדותי, מוגזם |
| מקצועי, אמין | יבש, אקדמי |
| חם, מעודד | קר, מרוחק |
| ברור, פשוט | מסובך, עמוס |
| WOW! "חייבים לנסות" | "חשוב ללמוד..." |

### ביטויים מותג

- "חידוש שחייבים להכיר"
- "ככה עובדים חכם"
- "AI שעושה את העבודה בשבילכם"
- "מהכיתה לעתיד"
- "30 שניות שישנו לכם את..."
- "הכלי שכל מורה צריך"

### פורמט Hook (פתיח)

| סוג | דוגמה |
|-----|-------|
| שאלה | "הידעתם ש...?" |
| WOW | "הכלי הזה משנה הכל!" |
| מספר | "3 טיפים ל..." |
| לפני/אחרי | "לפני: 3 שעות. אחרי: 30 שניות." |
| אתגר | "תוכלו לעשות את זה תוך דקה?" |

## מוזיקה

### סגנון
- ביטים אלקטרוניים רגועים
- Lo-fi / Chill hop
- לא אגרסיבי, לא חזק מדי
- מתאים לצפייה עם וגם בלי קול

### Volume
- מוזיקת רקע: 30% volume (כדי לא להפריע לכתוביות)
- Fade out ב-3 שניות האחרונות

### מקורות (royalty-free)
- Pixabay Music
- Mixkit
- YouTube Audio Library
