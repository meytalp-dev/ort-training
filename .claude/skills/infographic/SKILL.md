# Skill: אינפוגרפיקה – /infographic

כאשר המשתמש מבקש **ליצור אינפוגרפיקה** (סיכום ויזואלי בדף אחד) על כלי או נושא – הסקיל הזה מופעל.

---

## קלט

- תוכן מ-`/ai-expert` (מסמך חקר)
- או: נושא חופשי ("תעשה אינפוגרפיקה על Gemini")

---

## פלט

**קובץ HTML יחיד** – דף אחד שניתן ל:
- צפייה באתר
- הדפסה (A4 / A3)
- שיתוף בוואטסאפ (צילום מסך)
- שליחה בדוא"ל

שמירה ב: `docs/training/{tool-name}/infographic.html`

## עיצוב

### צבעים
- רקע: לבן (#ffffff) או בהיר מאוד (#f8fafb)
- כותרות: gradient כחול-ירוק (Lerani)
- אייקונים: צבעי הפלטה מ-`create-presentation` (תכלת, טורקיז, ירוק, אדום-ורוד)
- הדגשות: boxes עם רקע צבעוני בהיר

### פונטים
- כותרת: Secular One / Rubik Bold
- טקסט: Heebo 400
- מספרים: Poppins Bold

### גודל
- רוחב: 800px (מותאם הדפסה)
- @media print: מותאם A4

## תבנית HTML

```html
<!DOCTYPE html>
<html lang="he" dir="rtl">
<head>
  <meta charset="UTF-8">
  <title>{נושא} – אינפוגרפיקה | Lerani</title>
  <link href="https://fonts.googleapis.com/css2?family=Heebo:wght@300;400;500;700&family=Rubik:wght@500;700&family=Secular+One&display=swap" rel="stylesheet">
  <style>
    /* עיצוב מותאם הדפסה */
    @media print {
      body { margin: 0; }
      .no-print { display: none; }
    }
  </style>
</head>
<body>
  <div class="infographic">
    <!-- sections -->
  </div>
  <button class="no-print" onclick="window.print()">🖨️ הדפסה</button>
</body>
</html>
```
