# ספריית העיצובים של מייטל

כשמתחילים פרויקט חדש - בוחרים ביחד מהספרייה את הסגנון המתאים.

## איך משתמשים

1. מייטל אומרת מה היא צריכה (אתר / פוסט / מצגת / כו')
2. פותחים את הספרייה ובוחרים סגנון מתאים
3. מתאימים את הצבעים, הפונטים והאלמנטים לפרויקט

## הסגנונות

### 1. Warm Botanical (חם-בוטני)
- **תיקייה:** `warm-botanical/`
- **מתאים ל:** הזמנות, ספר בי"ס, תוכן למורים, חגים
- **מאפיינים:** רקע קרם, עלים/פרחים עדינים, צבעי אדמה
- **פונטים:** Playpen Sans Hebrew + Cafe + Heebo
- **דוגמה:** ספר "דברים מוכלאים" הרצל

### 2. Clean Management (ניהול נקי)
- **תיקייה:** `clean-management/`
- **מתאים ל:** מערכות ניהול, דשבורדים, טבלאות נתונים
- **מאפיינים:** רקע קרם, מסגרות צבעוניות, dusty rose hover
- **פונטים:** Shuneet3 Square Bold + Shuneet3 Medium
- **דוגמה:** מערכת הניהול הנוכחית

### 3. Educational Presentation (מצגת לימודית)
- **תיקייה:** `edu-presentation/`
- **מתאים ל:** שקפים לשיעור, חומרי הוראה
- **מאפיינים:** צבעים חיים אבל לא צורחים, אייקונים ברורים
- **פונטים:** Shuneet3 Square Bold + Petel Bold + Shuneet3 Medium
- **דוגמה:** מצגות הנושאים בבית הספר

### 4. Dramatic (דרמטי)
- **תיקייה:** `dramatic/`
- **מתאים ל:** היסטוריה, ספרות, אירועים מיוחדים
- **מאפיינים:** ניגודיות גבוהה, טיפוגרפיה חזקה
- **פונטים:** Antiochus Bold + Dybbuk + Heebo
- **דוגמה:** כרזת באר שבע

### 5. Playful (משחקי/ילדים)
- **תיקייה:** `playful/`
- **מתאים ל:** משחקים, פעילויות, תוכן לתלמידים
- **מאפיינים:** צבעים עליזים, פינות מעוגלות מאוד, אלמנטים כיפיים
- **פונטים:** Playpen Sans Hebrew + Cafe
- **דוגמה:** Genially, קלפי פרידה

### 6. Professional (מקצועי-רשמי)
- **תיקייה:** `professional/`
- **מתאים ל:** דוחות, מצגות להנהלה, מסמכים רשמיים
- **מאפיינים:** נקי, מינימלי, טיפוגרפיה קלאסית
- **פונטים:** Antiochus Bold + Shuneet3 Medium
- **דוגמה:** דוח אסטרטגי

## מקורות השראה מקנבה של מייטל

### לוח שנה עברי-לועזי 2025-2026 (DAGtmL7hrcA)
- **סגנון:** פרחי צבעי מים (watercolor wildflowers) על רקע קרם
- **כותרת על פתק נייר קרוע** - "לחלום. לתכנן. להגשים."
- **צבעים:** ורוד, סגול לבנדר, תכלת, ירוק מנטה, כתום עדין
- **מתאים ל:** כל תוצר warm botanical, שער חוברת, דף פתיחה

### מחברת מפקחים (DAGsGcZJTiI)
- **סגנון:** דומה ללוח השנה - פרחי צבעי מים מקיפים טקסט מרכזי
- **כותרת:** "לכתוב. לרשום. לשנות."
- **רקע לבן נקי** עם פרחים רק כמסגרת
- **מתאים ל:** שערי מחברות, ספר בי"ס, דפי פתיחה

### מגנטים "חלומות" (DAGO7uTg5t4)
- **סגנון:** Bold, צבעוני, גיאומטרי
- **טיפוגרפיה ענקית** עם צורות (כוכב, עיגולים, מטרה)
- **מתאים ל:** Playful - משחקים, מוטיבציה, תלמידים

### מילים טובות לתלמידים (DAGbVVCM9q8)
- **סגנון:** עיגולי צבעי מים (watercolor dots) בכל הצבעים
- **רקע לבן** עם כתמי צבע אורגניים
- **מתאים ל:** קלפים, סטיקרים, תעודות, אלמנטים דקורטיביים

### קלפי חמשת הממדים (DAGXTkpibqI)
- **סגנון:** כרטיסים צבעוניים עם אילוסטרציות
- **רקע צהוב/תכלת/ורוד** לכל קטגוריה
- **לוגו ORT במרכז** עם עיגול לבן
- **מתאים ל:** Playful - קלפים חינוכיים, משחקים

### קלפי שיח רגשי (DAG6_KqFfj0)
- **סגנון:** טקסטורת נייר, אילוסטרציה ילדותית, פרחים
- **רקע:** paper texture (#F5F0E8)
- **מתאים ל:** תוכן רגשי-חינוכי, ספר בי"ס

### קלפי דורית - פרח שקד (DAGbT_njvPA)
- **סגנון:** צבעי מים בוטני - פרח ורוד על רקע כתום-חם
- **מתאים ל:** אלמנטים דקורטיביים, תמונות רקע
