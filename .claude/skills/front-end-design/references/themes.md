# Themes – 10 ערכות עיצוב

כל theme מגיע עם: palette + fonts + אווירה + מתי להשתמש.

---

## 1. ORT (ברירת מחדל לכל תוצר אורט)

```css
--bg: #F0F4F8;
--primary: #2A9D8F;
--primary-dark: #1a7a6d;
--accent: #E9C46A;
--text: #0F172A;
```
**פונטים:** Heebo 800 + Rubik 400
**אווירה:** מקצועי, חינוכי, ים-ים-ים
**מתאים ל:** כל תוצר אורט – מצגות, דשבורדים, שאלונים

## איך לבחור Theme

```
תוכן טכני / נתונים      → Tech Dark
מצגת מקצועית            → Minimal Light
כל תוצר אורט            → ORT (ברירת מחדל)
היסטוריה / דרמה         → Midnight Galaxy
מדעים / טבע             → Botanical
שפה / ספרות             → Warm Editorial
עסקי / ניהולי           → Ocean Depths
יצירתי / אמנות          → Desert Rose
לא בטוח                 → ORT
```
