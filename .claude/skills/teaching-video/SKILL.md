# Skill: סרטון לימודי – /teaching-video

יצירת **סרטון לימודי אינטראקטיבי** – קובץ HTML שמדמה סרטון עם אנימציות, מעברים, שאלות ואוטו-פליי.

**המטרה:** המורה בוחר נושא ומקבל סרטון לימודי מלא שאפשר להציג בכיתה או לשלוח לתלמידים.

**זה לא מצגת רגילה.** זה סרטון שרץ לבד, עם אנימציות, שאלות אינטראקטיביות, ומעברים חלקים.

---

## קלט

המשתמש מספק:

| שדה | חובה | דוגמה |
|------|------|--------|
| מקצוע | כן | מתמטיקה, היסטוריה, אנגלית, מדעים, תספורת |
| נושא | כן | משוואות ריבועיות, המרד הגדול |
| כיתה | כן | ט, י, יא, יב |
| רמה | כן | בסיסי / בינוני / מתקדם |
| אורך | כן | קצר / בינוני / ארוך |
| סגנון | כן | מונפש / רגיל / מצחיק |

**אפשרויות קלט נוספות:**
- פרומפט חופשי ("תעשה סרטון על שטח משולש לכיתה ט")
- העלאת קובץ (PPTX / PDF / DOCX / TXT) – יחולץ התוכן ממנו

אם המשתמש לא ציין פרמטר – **לשאול אותו**.

## אורך הסרטון

| אורך | פריימים | זמן משוער | שימוש |
|-------|---------|-----------|-------|
| קצר | 8–12 | 2–3 דקות | חזרה מהירה, סיכום |
| בינוני | 15–20 | 5–7 דקות | הסבר נושא רגיל |
| ארוך | 25–35 | 10–15 דקות | נושא מורכב, הקנייה מלאה |

כל פריים מוצג **4–6 שניות** (auto-play) עם אפשרות לעצור ולנווט ידנית.

## סגנונות

### מונפש
- טקסט מופיע באנימציה (fadeIn, slideUp)
- אלמנטים נכנסים בזה אחר זה עם delay
- מעברים חלקים בין פריימים (fade)
- אייקונים מונפשים

### רגיל
- עיצוב נקי ומקצועי
- טקסט ברור עם הדגשות
- אייקונים תומכים
- מעברים פשוטים (fade)
- דומה למצגת מקצועית מתקדמת

### מצחיק
- הומור בטעם טוב – בדיחות קשורות לנושא
- טקסט בסגנון צעיר ושובב
- אנימציות אנרגטיות (bounce, shake)
- "הידעתם?" עם עובדות מפתיעות
- סגנון TikTok/YouTube Shorts

## מבנה הסרטון

### פריים 1 – פתיחה (Hook)
- כותרת הנושא בגדול
- משפט פתיחה שמושך תשומת לב
- סגנון מצחיק: "מה הקשר בין פיצה למשוואות?"
- סגנון רגיל: "היום נלמד על..."
- אנימציית כניסה

### פריימים 2–3 – למה זה חשוב?
- חיבור לחיים האמיתיים
- "למה בכלל צריך את זה?"
- דוגמה מהעולם של התלמידים

### פריימי הקנייה (הגוף)
- **מנות ידע קטנות** – רעיון אחד לפריים
- כל פריים: כותרת + הסבר קצר (2–3 שורות) + דוגמה
- הדגשות צבעוניות למושגים חשובים

### פריימי שאלה (כל 3–4 פריימים)
- שאלת הבנה קצרה
- "עצרו וחשבו" – pause אוטומטי
- אפשרות ללחוץ על תשובה (אינטראקטיבי)
- משוב מיידי (נכון/לא נכון + הסבר)

### פריים סיכום
- 3 נקודות מרכזיות
- "מה למדנו היום?"

### פריים סיום
- קריאה לפעולה ("נסו בעצמכם!")
- לוגו בית הספר / Lerani

## מבנה טכני – HTML

### Auto-Play
- הסרטון מתנגן אוטומטית (פריים כל 4–6 שניות)
- פס התקדמות (progress bar) בתחתית
- כפתורי שליטה: Play / Pause / הקודם / הבא
- ניווט מקלדת: חצים + רווח (pause/play)
- Touch/swipe לנייד

### מסך פתיחה
- "לחצו להתחיל" – overlay שמאפשר סנכרון עם הקלטת מסך
- אחרי לחיצה – auto-play מתחיל

### פריימים
```css
.frame {
  position: absolute;
  inset: 0;
  opacity: 0;
  transition: opacity 0.6s ease;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 2rem;
}
.frame.active {
  opacity: 1;
}
```

### אנימציות פנימיות
```css
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(30px); }
  to { opacity: 1; transform: translateY(0); }
}
.frame.active .animate {
  animation: fadeUp 0.5s ease forwards;
}
.frame.active .animate:nth-child(2) { animation-delay: 0.3s; }
.frame.active .animate:nth-child(3) { animation-delay: 0.6s; }
```

### שאלות אינטראקטיביות
- בפריימי שאלה – auto-play נעצר אוטומטית
- התלמיד בוחר תשובה → מקבל משוב → לוחץ "המשך"
- Auto-play חוזר אחרי הלחיצה

## עיצוב – מגניב, צעיר, בוגר

### כלל ברזל: לא ילדותי

- **כן:** מודרני, אנרגטי, צעיר – מתאים לנוער בתיכון
- **לא:** אימוג'ים מוגזמים, סגנון ילדים, צבעים צעקניים

### צבעים

- **אקסנט לפי מקצוע:** (אותה טבלה כמו ב-`create-video`)

| מקצוע | אקסנט |
|-------|--------|
| מתמטיקה | `#4F46E5` (כחול-סגול) |
| היסטוריה | `#B45309` (חום-זהב) |
| מדעים | `#059669` (ירוק) |
| עברית/תנ"ך | `#7C3AED` (סגול) |
| אנגלית | `#2563EB` (כחול) |
| תספורת | `#EC4899` (ורוד) |
| אזרחות | `#0891B2` (טורקיז) |
| ברירת מחדל | `#6366F1` (אינדיגו) |

- **רקע:** לבן (`#ffffff`) עם אלמנט עיצובי עדין
- **טקסט:** `#111` ראשי, `#666` משני

### טיפוגרפיה
- **כותרות:** `Rubik` weight 800-900
- **טקסט:** `Heebo` weight 400-600
- **מספרים:** `Secular One`
- Google Fonts: `Heebo:wght@300;400;700&family=Rubik:wght@400;700;900&family=Secular+One`

### יחס מסך
- ברירת מחדל: **16:9** (מסך מלא, מתאים להצגה בכיתה)
- `max-width: 960px; aspect-ratio: 16/9;`
- responsive לנייד

## כללים

1. **קובץ HTML אחד** – self-contained, בלי תלויות חיצוניות (חוץ מ-Google Fonts)
2. **RTL** – כיוון עברית, `dir="rtl" lang="he"`
3. **שפה פשוטה** – משפטים קצרים, מילים פשוטות
4. **מנות קטנות** – רעיון אחד לפריים, לא להעמיס
5. **שאלות** – כל 3–4 פריימים שאלת הבנה אינטראקטיבית
6. **Auto-play** – הסרטון רץ לבד, עם אפשרות לעצור
7. **מותאם לקהל** – תלמידי תיכון מקצועי עם קשיי קשב
8. **אם ניתן קובץ** – לחלץ תוכן ולבנות ממנו את הסרטון
9. **אם חסר פרמטר** – לשאול את המשתמש (לא לנחש)
10. **סגנון מצחיק** – הומור בטעם טוב, לא פוגעני, מתאים לנוער
11. **עיצוב מגניב** – לא ילדותי, צעיר, בוגר, מודרני
