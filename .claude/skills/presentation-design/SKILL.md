# Skill: עיצוב מצגת – /presentation-design

כאשר המשתמש מבקש **ליצור מצגת מעוצבת** או כשבונה העיצוב מייצר PPTX – יש להפעיל את הסקיל הזה.

# קלט

המשתמש יספק:
- **פלטת צבעים** (5 צבעים: primary, secondary, accent, bg, text)
- **שילוב פונטים** (כותרת + גוף)
- **סגנון** (אחד מ-4: geometric / soft / magazine / playful)
- **אופציונלי:** כותרות לשקפים, תוכן ספציפי

אם לא נבחר סגנון – לשאול.

# 4 סגנונות ויזואליים

## 1. גיאומטרי נועז (geometric)
**DNA ויזואלי:** חיתוכים אלכסוניים, בלוקי צבע גדולים (40-60% מהשקף), קצוות חדים, קונטרסט גבוה, צורות שגולשות מקצוות השקף.

עקרונות:
- חיתוך אלכסוני בשקף כותרת (rect מסובב 8-12 מעלות)
- בלוקי צבע מלאים (לא שקופים) שתופסים חצי שקף
- חלוקות א-סימטריות (55/45, לא 50/50)
- קווים מחברים (bridging bars) שחוצים בין אזורי צבע
- גבולות עבים בצד (לא למעלה) של כרטיסים

## 2. מודרני רך (soft)
**DNA ויזואלי:** פינות עגולות (radius 0.2-0.3), צללים רכים, שקיפויות גבוהות (85-93%), עיגולים וטבעות, הכל נושם.

עקרונות:
- roundRect עם radius גדול לכל אלמנט
- צללים: blur 15-20, offset 3-6, צבע 00000012
- עיגולי רקע גדולים מאחורי כרטיסים (transparency 80-85%)
- טבעות (oval עם fill שקוף + line) לעיטור
- נקודות (dot patterns) כטקסטורת רקע

## 3. מגזין עריכתי (magazine)
**DNA ויזואלי:** פסים רחבים (1.5-2") בצד, מספרים ענקיים כאלמנט ויזואלי, פריסה א-סימטרית, קווים דקים כאלמנטים מבניים.

עקרונות:
- פס צבע רחב (1.5-2") בצד ימין/שמאל (מתחלף בין שקפים)
- מספרי שקף ענקיים (60pt) בתוך הפס, בשקיפות 30%
- קו מחבר (bridging bar) שחוצה מתוכן אל הפס
- טקסט תמיד מיושר לצד אחד (לא ממורכז)
- קו דק (0.05") מפריד בין כותרת לתוכן

## 4. יצירתי ושובב (playful)
**DNA ויזואלי:** כרטיסים מוטים (rotate 1-3 מעלות), מוזאיקה א-סימטרית, נקודות ואשכולות עיגולים, גדלים מעורבים, סידור בקשת.

עקרונות:
- rotate: 1-2 מעלות על כרטיסים (טקסט נשאר ישר)
- מוזאיקה 2x2 לא סימטרית (שמאל-עליון גדול, ימין-עליון קטן)
- נקודות 3x3 (grid של עיגולים קטנים 0.12") כקישוט
- אשכולות: 3 עיגולים בגדלים שונים צמודים
- כרטיסים בגבהים שונים (קשת – אמצע גבוה מהצדדים)

# מבנה 10 שקפים

## שקף 1: כותרת (Title)
| סגנון | פריסה |
|-------|-------|
| geometric | חיתוך אלכסוני – rect מסובב, 60% צבע ראשי, טקסט על הגבול |
| soft | עיגול ענק (7") במרכז בצבע ראשי, כותרת בפנים, טבעות סביב |
| magazine | חצי שמאלי צבע ראשי מלא, כותרת בצד ימין על רקע בהיר |
| playful | כתמים אורגניים (3-4 עיגולים חופפים בזוויות שונות), כותרת מעל |

## שקף 2: אג'נדה / נקודות (Agenda)
| סגנון | פריסה |
|-------|-------|
| geometric | פסים אופקיים מדורגים (stagger), מספר בריבוע בקצה כל פס |
| soft | כרטיסים צפים עם צל, עיגול ממוספר ליד כל כרטיס |
| magazine | פס רחב בצד + 4 שורות טקסט עם קו מפריד דק ביניהן |
| playful | כרטיסים מוטים 1-2 מעלות, כל אחד בצבע אחר (transparency 85%) |

## שקף 3: 3 כרטיסים (Cards)
| סגנון | פריסה |
|-------|-------|
| geometric | 3 כרטיסים עם גבול שמאלי עבה (0.12"), בלוק צבעוני בראש כל כרטיס |
| soft | כרטיסים בקשת (אמצע גבוה), עיגולי רקע מאחורי כל כרטיס |
| magazine | 3 כרטיסים עם מספר ענק (42pt, transparency 20%) בפינה |
| playful | מוזאיקה (אחד גדול + 2 קטנים), כרטיסים מוטים |

## שקף 4: טקסט + תמונה (Content + Image)
| סגנון | פריסה |
|-------|-------|
| geometric | חלוקה 55/45, צד ימין בלוק צבע מלא עם placeholder תמונה |
| soft | כרטיס תוכן שמאל + placeholder תמונה עגולה (border-radius 50%) ימין |
| magazine | תמונה תופסת 60% רוחב, טקסט בפס צבעוני צר בצד |
| playful | תמונה מוטה 2 מעלות עם צל, טקסט לידה ישר |

## שקף 5: שתי עמודות (Two Columns)
| סגנון | פריסה |
|-------|-------|
| geometric | 2 בלוקים עם גבול עליון עבה בצבעים שונים, מופרדים בקו אנכי |
| soft | 2 פאנלים חופפים (overlap 0.5") כמו ון-דיאגרמה, יהלום בנקודת החפיפה |
| magazine | עמודה שמאל עם רקע צבעוני, עמודה ימין לבנה, קו מפריד דק |
| playful | 2 כרטיסים בגבהים שונים, הראשון מוטה ימינה והשני שמאלה |

## שקף 6: תמונה מלאה (Full Image)
| סגנון | פריסה |
|-------|-------|
| geometric | placeholder ממסגר עם שוליים 0.4", פס צבעוני תחתון עם כיתוב |
| soft | placeholder עם פינות עגולות + צל כבד + קווים כפולים מתחת |
| magazine | placeholder מלא עם כיתוב בפס צבעוני שחוצה את התמונה |
| playful | placeholder מוטה 1.5 מעלות עם צל, סטיקר-עיגול בפינה |

## שקף 7: תוכן עם פס צד (Side Accent)
| סגנון | פריסה |
|-------|-------|
| geometric | פס צבע מלא 1" בצד ימין, טקסט בצד שמאל, קו מחבר אופקי |
| soft | פס עגלגל (roundRect צר) בצד, עיגולים קטנים פזורים |
| magazine | פס רחב 2" עם מספר שקף ענק, טקסט מיושר ימינה |
| playful | פס בצורת גל (סימולציה עם עיגולים חופפים), נקודות צבעוניות |

## שקף 8: ציטוט / הדגשה (Quote)
| סגנון | פריסה |
|-------|-------|
| geometric | רקע צבע ראשי, משולש לבן בפינה, גרש ענק |
| soft | רקע צבע ראשי, טבעת זרקור (ring), גרש ענק כווטרמארק, גרדיאנט תחתון |
| magazine | רקע לבן, פס צבעוני רחב בצד, גרש ענק בתוך הפס |
| playful | רקע צבע ראשי, עיגולים צבעוניים פזורים, טקסט בתוך בועה |

## שקף 9: 4 תיבות (Grid)
| סגנון | פריסה |
|-------|-------|
| geometric | 2x2 grid סימטרי, כל תיבה עם גבול שמאלי בצבע אחר |
| soft | 4 כרטיסים עם צל, עיגול ממוספר בראש כל כרטיס, נקודות רקע |
| magazine | 2x2 מוזאיקה (שמאל-עליון גדול), מספרים ענקיים |
| playful | 4 כרטיסים מוטים בזוויות שונות, כל אחד בצבע רקע אחר |

## שקף 10: סיכום / תודה (Finale)
| סגנון | פריסה |
|-------|-------|
| geometric | רקע צבע ראשי, חיתוך אלכסוני הפוך משקף 1 (סימטריה) |
| soft | טבעות קונצנטריות (3), נקודות רקע, עיגולי אקסנט פזורים |
| magazine | חצי עליון צבע ראשי + חצי תחתון לבן, קו מפריד, טקסט סיכום |
| playful | קונפטי (20-30 עיגולים קטנים פזורים), טקסט "תודה!" ענק |

# כללי יצירה ב-PptxGenJS

## מידות שקף
- LAYOUT_WIDE: 13.33" x 7.5"
- כל המיקומים בinches

## טריקים חשובים

### חיתוך אלכסוני
```javascript
slide.addShape(pptx.ShapeType.rect, {
  x: 4.5, y: -1, w: 12, h: 10,
  fill: { color: priClr },
  rotate: -8
});
```

### טבעת (ring)
```javascript
slide.addShape(pptx.ShapeType.oval, {
  x: 3.5, y: 0.5, w: 6, h: 6,
  fill: { color: priClr, transparency: 100 }, // שקוף
  line: { color: accClr, width: 2, transparency: 40 }
});
```

### נקודות (dot pattern)
```javascript
for (let dx = 0; dx < 3; dx++) {
  for (let dy = 0; dy < 3; dy++) {
    slide.addShape(pptx.ShapeType.oval, {
      x: 12.0 + dx * 0.3, y: 0.3 + dy * 0.3,
      w: 0.12, h: 0.12,
      fill: { color: accClr, transparency: 50 }
    });
  }
}
```

### כרטיס מוטה
```javascript
slide.addShape(pptx.ShapeType.roundRect, {
  x: 1, y: 2, w: 3.5, h: 3,
  fill: { color: 'FFFFFF' },
  shadow: { type: 'outer', blur: 12, offset: 4, color: '00000015' },
  rectRadius: 0.18, rotate: 1.5  // הטיה קלה
});
// הטקסט נשאר ישר (ללא rotate)
slide.addText('כותרת', {
  x: 1.2, y: 2.3, w: 3.1, h: 0.5,
  fontSize: 18, fontFace: titleFont, color: txtClr,
  bold: true, align: 'center'
});
```

### שקיפויות מומלצות
| אלמנט | transparency |
|--------|------------|
| בלוק צבע (layout element) | 0 (מלא) |
| רקע כרטיס צבעוני | 85-93 |
| עיגול דקורטיבי | 70-85 |
| טבעת | line: 40-60 |
| מספר ענק (watermark) | 80-85 |
| גרש ציטוט | 85 |
| נקודות רקע | 50-60 |
| צל | blur 12-20, color '00000012' |

### גבולות (side borders)
```javascript
// גבול שמאלי עבה לכרטיס
slide.addShape(pptx.ShapeType.rect, {
  x: cardX + cardW - 0.12, y: cardY + 0.3,
  w: 0.12, h: cardH - 0.6,
  fill: { color: priClr }, rectRadius: 0.06
});
```

## ייחודיות – כל מצגת שונה

### Seeded PRNG
```javascript
function seededRandom(seed) {
  let s = seed;
  return function() {
    s = (s * 16807) % 2147483647;
    return (s - 1) / 2147483646;
  };
}
// Seed מהצבעים
const seed = parseInt(primary.slice(1,3),16) * 1000 +
             parseInt(secondary.slice(1,3),16) * 100 +
             Date.now() % 1000;
const rng = seededRandom(seed);
```

### סיבוב תפקידי צבעים
```javascript
const rotation = Math.floor(rng() * 3);
const roles = [
  { block: pri, accent: acc, deco: sec },
  { block: sec, accent: pri, deco: acc },
  { block: acc, accent: sec, deco: pri }
];
```

### וריאציות מיקום
```javascript
const decoX = 10 + rng() * 3;
const decoSize = 1.5 + rng() * 2;
const splitRatio = 0.40 + rng() * 0.15;
```

# Footer

כל שקף (חוץ מ-1 ו-10) כולל footer:
```javascript
function addFooter(slide, num, total) {
  slide.addShape(pptx.ShapeType.rect, {
    x: 0, y: 7.1, w: '100%', h: 0.4,
    fill: { color: priClr, transparency: 92 }
  });
  slide.addText(`${num} / ${total}`, {
    x: 0.3, y: 7.15, w: 1, h: 0.3,
    fontSize: 10, color: '999999', align: 'left'
  });
}
```
