# Skill: צור שאלון – /create-quiz

כאשר המשתמש מבקש **ליצור שאלון** על נושא כלשהו – יש להפעיל את הסקיל הזה.

---

# קלט חובה

המשתמש **חייב** לספק שלושה פרטים:

1. **נושא** – על מה השאלון
2. **כיתה** – ט / י / יא / יב
3. **רמה** – רמת הלימוד (3 יח"ל / 4 יח"ל / 5 יח"ל / בסיסי / מתקדם)

אם חסר אחד מהם – **שאל את המשתמש לפני שתמשיך**.

דוגמה לשימוש:
```
/create-quiz נושא: המרד הגדול | כיתה: י | רמה: 3 יח"ל
```

קלט אופציונלי נוסף:
- קובץ מצורף (PDF / DOCX / PPTX / טקסט) – הכלי ילמד את החומר וייצר שאלות ממנו
- מספר שאלות מותאם (ברירת מחדל: 10)

# מבנה השאלון

## 1. מסך פתיחה

- שם השאלון
- שדה להזנת שם התלמיד (חובה)
- שדה להזנת כיתה (חובה)
- כפתור "התחל"

## 2. שאלה מאבחנת (שאלה 1)

השאלה הראשונה היא **שאלה מאבחנת** ברמה בינונית.

מטרתה: לזהות את רמת התלמיד **מאחורי הקלעים**.

### חשוב מאוד – דיפרנציאליות סמויה

- **התלמיד לא יודע שהוא מאובחן**
- **אין שום סימון של "רמה" בממשק**
- **השאלון נראה זהה לכל התלמידים**
- הכל קורה בקוד ה-JavaScript ברקע

### לוגיקת האבחון

```
אם ענה נכון על השאלה המאבחנת → רמה גבוהה → שאלות מאתגרות יותר
אם ענה שגוי על השאלה המאבחנת → רמה בסיסית → שאלות פשוטות יותר
```

### יישום טכני

יש להכין **שתי ערכות שאלות** (אחרי השאלה המאבחנת):

- **ערכה א' (בסיסית):** שאלות של זיהוי, התאמה, שינון, הבנה ישירה
- **ערכה ב' (מתקדמת):** שאלות של יישום, ניתוח, השוואה, חשיבה

שתי הערכות על **אותו נושא** – רק ברמת קושי שונה.

שתי הערכות באותו אורך (אותו מספר שאלות).

**התלמיד לא רואה שום הבדל** – אותו עיצוב, אותו מספר שאלות, אותו ציון מ-100.

## 3. שאלות (2–10)

- שאלות אמריקאיות (4 אפשרויות)
- שאלה אחת בכל מסך
- אחרי בחירת תשובה: משוב מיידי (נכון ✓ / שגוי ✗)
- הסבר קצר לתשובה הנכונה
- כפתור "שאלה הבאה"

## 4. מסך סיום

- ציון סופי (מספר ואחוז)
- משוב כללי:
  - 90-100: מצוין!
  - 70-89: כל הכבוד!
  - 50-69: אפשר לשפר
  - 0-49: כדאי לחזור על החומר
- כפתור "שלח תוצאות"
- לאחר השליחה: הודעת אישור

# עיצוב השאלון

## סגנון כללי

- RTL עברית
- פונטים: Heebo + Rubik (Google Fonts)
- צבעים: אורט בית הערבה (תכלת, טורקיז, ירוק בהיר, לבן)
- עיצוב מודרני ונקי – כמו אפליקציה, לא כמו דף עבודה
- מתאים למובייל ולמחשב (responsive)

## CSS Variables

```css
:root {
  --bg:#F0F4F8;--card:#ffffff;
  --primary:#2A9D8F;--primary-l:#D4F3EF;--primary-d:#1a7a6d;
  --secondary:#E76F7A;--sec-l:#FDE8EA;
  --green:#52B788;--green-l:#D8F3DC;
  --dark:#1E293B;--gray:#64748B;
  --shadow:0 2px 20px -4px rgba(0,0,0,.06);
  --r:20px;
}
```

## רכיבים

- **כרטיס שאלה**: רקע לבן, פינות מעוגלות, צל עדין
- **כפתורי תשובה**: רקע בהיר, מסגרת, hover effect
- **תשובה נכונה**: ירוק (`--green-l` + `--green`)
- **תשובה שגויה**: ורוד (`--sec-l` + `--secondary`)
- **Progress bar** למעלה (כמה שאלות ענית מתוך הכל)
- **אנימציות**: fadeUp לכל שאלה חדשה

## מסך פתיחה

- כותרת גדולה עם שם השאלון
- אייקון מתאים לנושא
- שדות קלט מעוצבים (שם + כיתה)
- כפתור "התחל" בולט

- אייקון מתאים לציון
- הודעת משוב
