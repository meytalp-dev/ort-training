## תהליך — 5 שלבים

### שלב 1: הכנת טיוטה

לכל פוסט — להציג כרטיס טיוטה ברור:

```
┌─────────────────────────────────────┐
│  פלטפורמה: פייסבוק דף לרני         │
│  סוג: סרטון + טקסט                  │
│  קובץ: flow-video-full.mp4          │
│  טקסט: [הטקסט המלא]                │
│  סטטוס: ממתין לאישור                │
└─────────────────────────────────────┘
```

כללים:
- טיוטה אחת = פוסט אחד. לא חבילות.
- להציג את הטקסט המלא, לא תקציר.
- אם יש סרטון — לציין את שם הקובץ המדויק.
- אם יש תמונה — להציג אותה.

### שלב 2: אישור מייטל

לחכות לאישור מפורש. אם מייטל אומרת:
- "כן" / "פרסמי" / "יאללה" → להמשיך לשלב 3
- "שני" / "תתקני" → לתקן ולהציג שוב
- "לא" → לעצור

### שלב 3: פרסום לפי סדר פלטפורמות

| סדר | פלטפורמה | מי מפרסם | כלי |
|------|-----------|----------|------|
| 1 | פייסבוק דף לרני | אני (API) | fb_publish_video / fb_publish_photo / fb_publish_post |
| 2 | אינסטגרם | אני (API) | ig_publish_reel / ig_publish_photo |
| 3 | קבוצת לרני (FB) | מייטל (ידנית) | אני מכין טקסט להעתקה |
| 4 | פרופיל אישי (FB) | מייטל (ידנית) | שיתוף מהדף |
| 5 | לינקדין | מייטל (ידנית) | אני מכין טקסט להעתקה |

**חשוב:**
- לפרסם פלטפורמה-פלטפורמה, לא הכל ביחד
- אם סרטון — צריך URL ציבורי. להעלות קודם ל-GitHub Pages:
  1. להעתיק ל-`docs/marketing/assets/posts/`
  2. להוסיף חריגה ב-`.gitignore`
  3. לדחוף ל-GitHub
  4. URL: `https://meytalp-dev.github.io/ort-training/marketing/assets/posts/{filename}`

### שלב 4: בדיקה אחרי פרסום

אחרי כל פרסום, לבדוק:
- [ ] הסרטון/תמונה עלו ומוצגים
- [ ] הטקסט מופיע מלא (לא חתוך)
- [ ] אין כפילויות
- [ ] לדווח למייטל עם לינק ישיר לפוסט

להשתמש ב-`fb_get_posts` ו-`ig_get_posts` לאימות.

### שלב 5: תיעוד

אחרי פרסום מוצלח:
1. לעדכן את הדשבורד (`docs/marketing/social-media-hub.html`) — סטטוס published
2. לעדכן את רשימת הפרסומים בזיכרון (`project_published_posts.md`)
3. לעדכן Link in Bio (`docs/marketing/linkinbio.html`) אם רלוונטי
4. **אם הפוסט הוא טיפ יומי** — להוסיף כרטיס לעמוד הטיפים באתר Learni:
   - קובץ: `docs/index.html` — סקשן `page-tips`, בתוך ה-`div#tipsGrid`
   - להוסיף בלוק `tip-item` חדש **בתחילת** ה-`tipsGrid` (הטיפ החדש ביותר למעלה)
   - לעדכן: תאריך, כותרת, תמונה (URL מ-GitHub Pages), צבע כלי, שם כלי
   - לדחוף ל-GitHub
   - תבנית בלוק:
